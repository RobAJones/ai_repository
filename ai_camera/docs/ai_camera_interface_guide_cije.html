Camera Controls

Stop/Start Stream — Toggles live MJPEG feed from the OV3660 camera.

Capture — Saves a still frame to SD card as IMG_XX.jpg.

Review — Browse previously captured images on the SD card.

Audio Controls

Record — Records from the onboard PDM mic to REC_XX.wav on SD.

Speaker — Enables/disables TTS playback via MAX98357A amp.

Playback Bar — Preview any saved WAV file before sending to OpenAI.

File Management

Add to Export — Queues the selected file for the next OpenAI request.

Save to PC — Downloads the selected file from SD to your computer.

Delete File — Permanently removes file from SD card (irreversible).

Analysis & Export

Send to OpenAI — Submits image + audio prompt to GPT-4 Vision / Whisper API.

Replay TTS — Replays the last Text-to-Speech response via speaker.

Export Package — Downloads a ZIP with image, audio, and analysis text.

Workflow Tips

Follow the Pairing Workflow banner: Capture → Export → Send.

Up to 5 images can be queued simultaneously for multi-image GPT-4 Vision analysis.

Analysis takes 60–90 seconds — the page will update automatically when complete.

Hardware Notes

Camera: OV3660, 160° wide-angle, IR LEDs (GPIO 47).

Mic: PDM on GPIO 38/39 — shares I2S_NUM_0 with camera; blocking record only.

SD card: SD_MMC on GPIOs 11/12/13 — stores images, audio, and exports.
