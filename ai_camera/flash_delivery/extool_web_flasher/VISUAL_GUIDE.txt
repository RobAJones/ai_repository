STEP 1: Open Arduino IDE
    │
    ├── Load: ESP32_AI_Camera_v3_0_0.ino
    ├── Configure: ESP32S3 Dev Module settings
    └── Export: Sketch → Export Compiled Binary
         │
         ├─→ Generates 3 files in build/ folder:
         │    • ESP32_AI_Camera_v3_0_0.ino.bootloader.bin
         │    • ESP32_AI_Camera_v3_0_0.ino.partitions.bin
         │    • ESP32_AI_Camera_v3_0_0.ino.bin
         │
         └─→ Find 1 more file in Arduino15/ folder:
              • boot_app0.bin


STEP 2: Copy & Rename
    │
    └── Copy to web_flasher_package/firmware/:
         │
         ├── ESP32_AI_Camera_v3_0_0.ino.bootloader.bin  →  bootloader.bin
         ├── ESP32_AI_Camera_v3_0_0.ino.partitions.bin  →  partitions.bin
         ├── ESP32_AI_Camera_v3_0_0.ino.bin             →  firmware.bin
         └── boot_app0.bin                              →  boot_app0.bin


STEP 3: Test
    │
    ├── Run: start_server.bat (Windows) or start_server.sh (Mac/Linux)
    ├── Open: http://localhost:8000
    ├── Connect: ESP32-S3 via USB
    └── Click: "Install AI Camera Firmware"


STEP 4: Deploy (Optional)
    │
    ├── GitHub Pages: Upload to repo, enable Pages
    ├── Netlify: Drag & drop folder
    └── Share URL with users!
