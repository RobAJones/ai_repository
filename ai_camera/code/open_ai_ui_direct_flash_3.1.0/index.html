📋 Installation Steps

1

Connect Hardware

Plug your ESP32-S3 AI Camera into your computer using a USB-C data cable. Make sure the cable supports data transfer, not just charging.

2

Click Install Button

Click the "Install AI Camera Firmware" button above. Your browser will ask to select a serial port - choose the one that represents your ESP32 device.

3

Flash Firmware

The firmware will be automatically downloaded and flashed to your device. This takes about 2-3 minutes. Don't disconnect during flashing!

4

Setup Wizard

After flashing, the device creates a WiFi network called "AI-Camera-Setup". Connect to it from your phone or laptop to configure WiFi credentials and OpenAI API key.

5

Start Using!

Once configured, the device connects to your WiFi. Open the IP address shown in the web interface to start using your AI camera!

💡 Tip: After installation, you can factory reset the device by holding the BOOT button for 3 seconds. This brings back the setup wizard.
🔧 Troubleshooting: If the install button doesn't work, make sure you're using Chrome, Edge, or Opera browser. Firefox and Safari don't support WebSerial yet.
